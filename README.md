## Formato de cada writeup

Cada lab tem seu próprio arquivo `.md` com:

- O que a vulnerabilidade é e por que ela existe
- Como resolvi passo a passo
- O payload usado
- O que tentou me bloquear (filtros, WAF)
- Como isso aparece em ambientes AWS reais
- O que aprendi
